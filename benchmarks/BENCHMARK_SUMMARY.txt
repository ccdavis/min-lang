┌─────────────────────────────────────────────────────────────────────┐
│              OPTIMIZATION OPPORTUNITIES                              │
└─────────────────────────────────────────────────────────────────────┘

To match Python performance (~4× speedup):
  1. Specialized numeric instructions (20-30% gain)
  2. Register-based VM instead of stack (30-50% gain)
  3. Inline caching for field access (10-20% gain)
  4. Float-specific fast paths (30-40% gain)

To approach Go performance (~75× speedup):
  1. JIT compilation for hot loops
  2. Type specialization
  3. Escape analysis and stack allocation
  4. SIMD vectorization
